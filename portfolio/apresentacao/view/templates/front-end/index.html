Front-end

1. Estrutura HTML e Elementos Básicos HTML (HyperText Markup Language): Linguagem de marcação usada para criar a estrutura de páginas web. Estrutura Básica: Inclui elementos principais como: : Define o início do documento HTML. : Contém metadados, como e links para estilos e scripts. <body>: Onde o conteúdo visível da página é definido. Elementos Comuns: Cabeçalhos: <h1> a <h6>, para títulos e subtítulos. Parágrafos: <p>, para textos comuns. Imagens: <img>, para exibir imagens. Links: <a>, para criar hiperlinks. 2. Formulários e Validações Formulários: Utilizados para coletar dados do usuário. Estrutura básica: <form>: Envolve os campos de entrada. Campos de entrada: <input>, <textarea>, <select>. Botões: <button> ou <input type="submit">, para enviar os dados. Validações: Validação HTML: Usando atributos como required, minlength, maxlength, pattern para controlar a entrada de dados. Validação com JavaScript: Personalização das regras de validação, exibindo mensagens dinâmicas. 3. Estilização com CSS CSS (Cascading Style Sheets): Usado para estilizar o conteúdo HTML. Seletores: Identificam os elementos que serão estilizados (ex: h1, .classe, #id). Propriedades Comuns: Cor: color, background-color. Tamanho e Espaçamento: font-size, margin, padding. Posicionamento: position, display, float. 4. Pseudo-classes e Personalização de Inputs Pseudo-classes: Permitem alterar o estilo de um elemento com base no seu estado: :hover: Aplicado quando o usuário passa o mouse sobre o elemento. :focus: Quando um elemento (como um campo de formulário) está em foco. :nth-child(): Para estilizar elementos com base na sua posição na hierarquia. Personalização de Inputs: Estilos personalizados para campos de formulário, botões e elementos interativos usando propriedades como border, outline, background e ajustando o comportamento com pseudo-classes (:checked, :disabled). 5. Introdução ao JavaScript JavaScript: Linguagem de programação que adiciona interatividade às páginas web. Funções Básicas: Manipulação do DOM (Document Object Model) para alterar dinamicamente o conteúdo da página (document.getElementById, document.querySelector). Manipulação de eventos como click, submit, keydown para criar interatividade. Uso de variáveis, condicionais e loops para controlar o comportamento da página. Integração com HTML/CSS: JavaScript pode alterar estilos, adicionar ou remover elementos, validar formulários e muito mais. </body> </html>
